# Research Paper Presentation Website
## Project: Small Language Models + OCR for Medical Diagnostics

This repository contains a sleek, modern, single-page presentation website for the research paper "Small Language Models + OCR for Medical Diagnostics - A Systematic Review".

### Features
- **10-Minute Presentation Timer**: Auto-starts on load, with color-coded warnings (Yellow at 8m, Red at 10m).
- **Glassmorphic Design**: Modern dark theme with semi-transparent cards and backdrop blur effects.
- **Interactive Charts**: Performance and cost comparisons using Chart.js.
- **Smooth Navigation**: Fixed navbar with active section highlighting.
- **Animations**: Subtle scroll animations using AOS.
- **Responsive Layout**: Optimized for Desktop, Tablet, and Mobile.

### Project Structure
- `index.html`: Main structure and content.
- `styles.css`: Custom styling and design tokens.
- `script.js`: Interactive logic, timer, and charts.
- `assets/`: 
    - `image1.jpeg`: Architecture Flowchart
    - `image2.jpeg`: Preprocessing Pipeline
    - `image3.jpeg`: Performance Comparison
    - `image4.jpeg`: OCR Timeline

### Deployment to GitHub Pages
1. Create a new repository on GitHub named `slm-ocr-medical-review-presentation`.
2. Push all files (`index.html`, `styles.css`, `script.js`, and the `assets` folder) to the main branch.
3. Go to **Settings** -> **Pages**.
4. In the **Build and deployment** section, select **Deploy from a branch** and Choose **main** / **(root)**.
5. Click **Save**.
6. Your website will be live at `https://yourusername.github.io/slm-ocr-medical-review-presentation/` in a few minutes.
