Pipeline Architectures

Architecture A

OCR-then-NER

Standard sequential pipeline. Optimized for latency but prone to error propagation.

Architecture B

Vision-Language

Direct image-to-text transformation. Bypasses OCR errors but requires high GPU VRAM.

Architecture C (REC)

Hybrid Multi-Engine

Voting mechanism + Human-in-the-loop. Achieves 99%+ accuracy with safety safeguards.

TL;DR: Key Takeaways

1. Hybrid Strategy is Key

SLMs for Extraction, LLMs for Reasoning.

2. Cost Savings

SLMs can save $7.3M/year for 1M daily queries.

3. Architecture C wins

Multi-engine voting + Human-in-the-loop = Safety.

4. Human Oversight

Non-negotiable requirement for clinical safety.
